## 📦 Features

---

## 🚀 How to Install

### 1️1. Clone this repo
```bash
git clone https://github.com/jihan431/my-dotfiles.git
cd my-dotfiles
```
### 2. Launch install script
```bash
chmod +x install.sh
./install.sh
```
